### 08 | 开发 | 创建 wechat-local-chat

新建目录 `wechat-local-chat/`：

| 文件 | 作用 |
|------|------|
| `index.mjs` | 主桥接：iLink 收消息 → Ollama `/api/chat` → 回发微信 |
| `config.json` | 模型、Ollama 地址、system prompt、历史条数 |

关键逻辑：
- 复用 `cli-in-wechat/dist/ilink/` 的登录与长轮询
- 凭证目录：`~/.wechat-local-chat/`（与 CLI 桥接隔离）
- 支持 `/new`、`/help` 命令
- 每用户独立对话历史，最多保留 20 轮

```powershell
# 冒烟测试

# → Ollama 连接成功，二维码正常显示
```

**结果：** 桥接可启动；待用户微信扫码完成端到端验证
