# 微信截图发送 (WeChat Screenshot)

## 触发词
- 截图
- 截屏

## 能力
截取电脑屏幕并通过微信发送图片给用户。

## 实现步骤

1. 运行 `scripts/screenshot.ps1`（必须通过 PowerShell 运行）
2. 脚本会自动截屏 → 启动 Python HTTP 服务器 → 发送到微信 → 清理
3. 等待脚本输出 `WECHAT_OK: 截图已发送` 后回复用户

## 注意事项
- 需要 Python 环境（用于启动 HTTP 服务器）
- 脚本必须在 30 秒内退出
- 发送成功后不要再用文字描述图片内容
- 如果用户说"没收到"，再运行一次即可
